# SIMPLE QUIZ APP

- To run the app, first install Flask with:
`pip install flask`
or
`python3 -m pip install flask`
- Then, simply type: `python app.py` or `python3 app.py` (Linux)

> [!NOTE]
> To enter a quiz. Type in path: /chap/{chap_num} with chap_num from 1 to 9
